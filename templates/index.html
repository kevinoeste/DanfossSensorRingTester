Sensor Ring Test Program Functions


Start Test:
To perform a test click on the start test tab. Enter the serial number of the sensor ring and then click start test.
The test will begin and once complete the results will be displayed along with if it passed or failed.


View Data:
This will display the data from every completed test. The search function allows all results to be displayed
based on the serial number. All data in the database is saved to the local PC.


Edit Database:
If needed data can be manually entered into the database or a previous test can be edited. Based on the data that is entered
it will determine if the sensor passed or failed.


Info:
Information about the fixture, test parameters, etc.
